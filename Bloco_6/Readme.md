## HTML & CSS - Forms

### Você será capaz de:
- Criar formulários em HTML com as tags: input, button, textarea, select, form.

```html
    <form action='minhapage.html' method='post'>
        <!-- GET = pela url -->
        <!-- POST = escondido -->
        <input
            required='required'
            type='text'
            name='email'
            id='input-text'
            class='form-blue input'
            placeholder='digite seu email'>
        <button type='submit'>Enviar</button>
    </form>
```
